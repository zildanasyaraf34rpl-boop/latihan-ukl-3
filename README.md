Program ini bertujuan untuk menghitung nilai faktorial dari sebuah bilangan bulat n. Faktorial (ditulis n!) adalah hasil perkalian semua bilangan bulat positif dari 1 sampai n.

Cara Kerja Program:
Input: Program meminta pengguna memasukkan satu bilangan bulat (n).
Validasi: Program mengecek apakah bilangan itu negatif (n < 0). Jika ya, program berhenti karena faktorial tidak terdefinisi untuk bilangan negatif.
Inisialisasi long: Program menyiapkan variabel hasilFaktorial dengan nilai awal 1. Tipe datanya long (bukan int) agar bisa menampung angka yang sangat besar (faktorial tumbuh sangat cepat).
Perulangan (for loop): Program melakukan perulangan (looping) sebanyak n kali.
Output: Setelah loop selesai, variabel hasilFaktorial akan berisi nilai faktorial yang benar, yang kemudian ditampilkan ke layar.
